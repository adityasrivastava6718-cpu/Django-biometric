# Django Biometric Attendance System

This project is a **Biometric Attendance Management System** developed using **Python and Django**.

The system is designed to process biometric machine attendance data provided in **.dat file format**, store employee attendance punch logs, and serve as a base for generating attendance and exception reports.

---

## Features
- Employee management using Django Admin
- Storage of raw biometric attendance punch logs
- Database-backed attendance tracking
- Admin panel for managing employees and attendance data
- Scalable structure for attendance calculation and reporting

---

## Technologies Used
- Python
- Django Framework
- SQLite Database
